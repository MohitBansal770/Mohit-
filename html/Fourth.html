Using CSS with HTML

Css is used to change physical attributes like color, size, background etc. Styling can be done in HTML itself using style modifier inside the tag or linking style sheets to the HTML.

What is Css?

  • CSS stands for Cascading Style Sheets.
  • CSS describes how HTML elements are to be displayed on screen, paper, or in other media.
  • CSS saves a lot of work. It can control the layout of multiple webpages all at once
  • External stylesheets are stored in CSS files

We can have multiple design skews for same page and make them look diffrent using diffrent css sheets on similar HTML pages.
